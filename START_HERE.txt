🚀 DEPLOYMENT OPTIONS:

OPTION 1: GitHub + Netlify (Recommended - Easy Auto-Deploy)
─────────────────────────────────────────────────────────────
1. Create GitHub repo at https://github.com/new
2. In Terminal, run these commands:
   
   cd /Users/alvarogonzalez/Desktop/ssd-website-deploy
   git init
   git add .
   git commit -m "Initial commit - SSD website"
   git remote add origin YOUR_GITHUB_REPO_URL
   git push -u origin main

3. Go to https://app.netlify.com
4. Click "Add new site" → "Import an existing project"
5. Choose GitHub → Select your repo
6. Click "Deploy site" (Netlify auto-detects Next.js!)


OPTION 2: Netlify CLI (Quick Testing)
─────────────────────────────────────
1. Install Netlify CLI:
   npm install -g netlify-cli

2. Login:
   netlify login

3. Deploy:
   cd /Users/alvarogonzalez/Desktop/ssd-website-deploy
   netlify init
   netlify deploy --prod
